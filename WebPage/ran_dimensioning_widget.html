RAN Dimensioning

This widget estimates the number of RAN sites required to comply with the coverage target polygons. It considers the coverage and capacity network considerations.

In order to use this widget, perform the following steps:

  • Introduce the network design characteristics in the "General Inputs" section. Use the defined values from the network architecture.
  • Introduce the coverage target polygons' characteristics in the "Polygon Inputs" section. Use the output values from Tribal Mapping Tool.
  • (Optional for Advaced Users Only) Modify the "Advance Inputs" sections to further customize the calculations.
  • Click on the "Calculate" button to perform the calculations.
  • The calculation's result can be found in the "Results" section.

Basics for this widget are addressed on the Network Planning & Design Module.
